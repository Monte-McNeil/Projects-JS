CSS or Cascading Style sheets is used to style the HTML document, it describes how HTML elements should be displayed.
It save a bunch of time, it controls the layout of multiple web pages at once in 1 seperate file

Comments are also used for multi-line: /* */

CSS syntax contains a selector, and a declaration block

simple selector:

p {text-align: center;}

id selector

#id-selector {color: red;}

class selector

.class-selector {color: red;}

A specific class selector

p.class-selector{color: red;}

Universal selector, useful for getting rid of pre-made default styles browsers use

* {background-color: red;}

Grouping selectors are good for multiple elements you want the same style for

h1, h2, p{color:red}

Margins

  • margin-top
  • margin-right
  • margin-bottom
  • margin-left

There are also different values you can give these

  • auto - the browser calculates the margin
  • length - specifies it in px, pt, cm, etc
  • % - specifies the width in percentage from its containing element
  • inherit - specifies the margin should be inherited from its parent element

There is a shorthand too, so margin: t r b l;

Padding

Just like margins there is top, right, bottom, left. But the values for these can be length, %, or inherit

The shorthand is the same as margins too

The box-sizing property is usefull when dealing with width and padding working together say you have an element with a width of 300px and a padding of 25px the actuall width of that element will be 400px. Using bos-sizing: border-box; it keeps the width at 300px no matter what.
